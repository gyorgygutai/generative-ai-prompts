**Core Rules**
- Fixed Semantic Blocks: Subject/Pose | Body | Bust/Chest | Clothing | Environment/Lighting | Camera/Quality
- Objective Spatial Language: Proportions relational, zero subjective adjectives
- Light-Material Coupling: Surface type + light behavior + texture response per block
- Physics-Aware Tension: Weight, pressure, gravity, and contact surface logic explicit
- Layered Skin Sequence: Silhouette → tone contrast → pores/veins → vellus → sheen
- Framing Lock: Crop boundaries, height, angle, DOF falloff, focus priority
- Anchor Tokens (Max 3): Structural/metallic details only
- Constructive Absence: State what occupies empty space
- Validation Rule: If token doesn't answer Where/Shape/Material/Light/Hold/See/Absent → cut it
